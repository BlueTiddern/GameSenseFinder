API: Application programming interface (API): Acts like a bridge between two pieces of software so that different systems can interact with each other.

REST and SOAP APIs

SOAP

- simple object access protocol
- uses a service interface
- Is a protocol
- Only uses XML 
- Is like an envelope
- Req more server and bandwidth
- Supports WS-security
- Has a web service Description language file
- SOAP API exposes the operation

REST
- Representational state transfer
- Uses URIs
- Is an archi style
- uses json, XML
- Like a post card
- Rest API exposes the data

PostMan: API platform for building and using APIs. It simplifies each step of API lifecycle and streamlines colaboration.
